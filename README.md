# Petshop react project
## Requirements
install nodejs >= 9
### Install dependencies:
`npm install`

## How to run
### Run react(port 3000) and json-server(port 3004):
`npm start`

## How to build
### Generate the build:
`npm run build`
### Run the json server(mock):
`json-server --watch db.json --port 3004`
### You may serve the build with a static server:
`npm install -g serve`
###
`serve -s build`
